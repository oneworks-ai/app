<h1 align="center">One Works 万可</h1>

<p align="center"><strong>一个面向 AI 工作流的工作空间。</strong></p>

## 介绍

One Works（万可）是一个开源 AI 工作空间，核心是可扩展的插件体系和标准化配置体系。它统一管理 agents、工具、适配器、会话和本地运行数据，覆盖桌面端、Web、VS Code 和 CLI，让你一次配置，在所有入口中使用同一套 AI 工作流。

## 快速开始

```bash
npx oneworks
```

<details>
<summary>安装方式</summary>

```bash
npm install --global oneworks
```

Homebrew、Scoop 和桌面端构建产物会在公开包构建可用后由发布自动化生成。

</details>

## 简单用例

```bash
npx oneworks web
npx oneworks summarize this repository
```
