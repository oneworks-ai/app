# Neo Workshop Theme

An optional, separately published OneWorks theme plugin. It provides a neo-brutalist recipe with cream surfaces,
black structural borders, hard offset shadows, yellow highlights, and playful pink actions. The core app does not
bundle or enable it by default.

```bash
pnpm add -D @oneworks/plugin-neo-workshop-theme
```

```json
{
  "plugins": [
    {
      "id": "@oneworks/plugin-neo-workshop-theme",
      "scope": "neo-workshop-theme"
    }
  ]
}
```

Select **Neo Workshop** under **Settings → Themes**.
