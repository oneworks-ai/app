# 新粗野工坊主题

这是一个独立发布、由用户选择安装的 OneWorks 主题插件，提供奶油色表面、黑色结构边框、硬偏移阴影、
黄色高亮与俏皮粉色操作组成的新粗野主义配方。核心应用不会内置或默认启用它。

```bash
pnpm add -D @oneworks/plugin-neo-workshop-theme
```

```json
{
  "plugins": [
    {
      "id": "@oneworks/plugin-neo-workshop-theme",
      "scope": "neo-workshop-theme"
    }
  ]
}
```

之后可在 **设置 → 主题** 中选择 **新粗野工坊**。
