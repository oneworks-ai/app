# Codex 主题

这是一个独立发布、由用户选择安装的 OneWorks 主题插件，提供中性层级、纤细分隔线、紧凑控件与克制蓝色
交互反馈组成的低噪工作台。核心应用不会内置或默认启用它。

```bash
pnpm add -D @oneworks/plugin-focus-workbench-theme
```

```json
{
  "plugins": [
    {
      "id": "@oneworks/plugin-focus-workbench-theme",
      "scope": "focus-workbench-theme"
    }
  ]
}
```

之后可在 **设置 → 主题** 中选择 **Codex 主题**。
