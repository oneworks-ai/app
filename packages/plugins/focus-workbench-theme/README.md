# Codex Theme

An optional, separately published OneWorks theme plugin. It provides a restrained, low-noise workbench with neutral
layers, thin dividers, compact controls, and quiet blue interaction feedback. The core app does not bundle or enable
it by default.

```bash
pnpm add -D @oneworks/plugin-focus-workbench-theme
```

```json
{
  "plugins": [
    {
      "id": "@oneworks/plugin-focus-workbench-theme",
      "scope": "focus-workbench-theme"
    }
  ]
}
```

Select **Codex** under **Settings → Themes**.
