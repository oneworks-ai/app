# Cowork 主题

这是一个独立发布、由用户选择安装的 OneWorks 主题插件，提供象牙白制图网格、宽松卡片、珊瑚色主操作、
蓝色进度语义与 9/14/20 圆角阶梯。核心应用不会内置或默认启用它。

```bash
pnpm add -D @oneworks/plugin-warm-cowork-theme
```

```json
{
  "plugins": [
    { "id": "@oneworks/plugin-warm-cowork-theme", "scope": "warm-cowork-theme" }
  ]
}
```

之后可在 **设置 → 主题** 中选择 **Cowork 主题**。
