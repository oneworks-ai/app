# Cowork Theme

An optional, separately published OneWorks theme plugin. It provides an ivory drafting grid, roomy cards, coral
primary actions, blue progress semantics, and a 9/14/20 radius ladder. The core app does not bundle or enable it by
default.

```bash
pnpm add -D @oneworks/plugin-warm-cowork-theme
```

```json
{
  "plugins": [
    { "id": "@oneworks/plugin-warm-cowork-theme", "scope": "warm-cowork-theme" }
  ]
}
```

Select **Cowork** under **Settings → Themes**.
