# One Works Docs

en-US | [zh-Hans](./README.zh-Hans.md)

This directory is the modular documentation area for external README content.

Root-level public documentation lives in the homepage docs site:
[oneworks-ai.github.io/docs](https://oneworks-ai.github.io/docs/).

Use this directory only when a repository-local README needs to split external user-facing usage details into smaller files. Internal module organization, maintenance rules, and agent memory belong in `AGENTS.md` or `.oo/rules/`.
