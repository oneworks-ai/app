# One Works 0.1.0-beta.7

- Align every workspace package on `0.1.0-beta.7` so the beta channel resolves one coherent release across apps, runtimes, adapters, channels, and plugins.
- Add adapter-owned imports for model services and worktree environments, including Codex user and project `config.toml` providers, source-aware environment migration, and searchable adapter selection.
- Add extensible client theme plugins with scoped runtime registration, localized settings, safe fallback behavior, and installable China Red, Focus Workbench, Neo Workshop, and Warm Cowork themes.
- Add external Chrome control through the OneWorks extension, including explicit pairing, semantic tab and frame operations, bounded advanced access, deterministic release artifacts, and Chrome Web Store delivery.
- Improve managed computer and in-app browser control with workflow execution, cursor configuration, safer scheduling, action-state indicators, and clearer recovery behavior.
- Unify plugin and skill discovery across supported adapters, add searchable marketplaces, and harden installation with transactional updates and rollback.
- Add a configurable chat history timeline, refine the starter composer and navigation surfaces, and improve Codex effort and account-status controls.
- Improve Relay login, server discovery, project rules, team document synchronization, and configuration editing.
